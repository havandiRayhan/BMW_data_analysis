🚗 BMW Market Data Analysis Project

📌 Deskripsi Proyek

Proyek ini bertujuan untuk melakukan analisis mendalam terhadap pasar mobil bekas BMW menggunakan dataset sintetis. Fokus utama dari analisis ini adalah untuk memberikan rekomendasi strategis bagi bisnis dealer mobil mewah dalam mengoptimalkan inventaris, menentukan harga jual, dan memahami faktor-faktor yang mempengaruhi depresiasi harga.

📊 Dataset

Dataset yang digunakan adalah bmw_cars_market_dataset_synthetic.csv.
Fitur utama dalam dataset ini meliputi:

Model: Seri mobil BMW (misal: 1 Series, X5, M3).

Year: Tahun produksi kendaraan.

Mileage: Jarak tempuh kendaraan (km).

Price: Harga pasar saat ini (USD).

Accident History: Riwayat kecelakaan kendaraan.

Fuel Type & Emission: Jenis bahan bakar dan tingkat emisi CO2.

🎯 Pertanyaan Bisnis (Case Study)

Dalam simulasi ini, analisis difokuskan untuk menjawab pertanyaan dari sisi manajemen (Executive Perspective):

Analisis Depresiasi: Bagaimana pengaruh mileage terhadap harga di berbagai model?

Dampak Riwayat Kecelakaan: Seberapa besar penurunan harga pada unit yang pernah mengalami kecelakaan?

Optimasi Regional: Negara mana yang memberikan profit tertinggi untuk kategori SUV?

Tren Bahan Bakar: Apakah mobil Hybrid memiliki nilai jual kembali yang lebih baik dibanding bensin/diesel?

Rekomendasi Investasi: Model apa yang paling menguntungkan untuk dibeli dengan budget tertentu?

🛠️ Teknologi yang Digunakan

Bahasa Pemrograman: Python 3.13.5

Library Utama:

Pandas: Untuk manipulasi dan pembersihan data.

Matplotlib: Untuk visualisasi data dasar.
